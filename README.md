# AI-Based Electricity Theft Detection

## Project Overview
This project implements an anomaly detection system using Isolation Forest algorithm to detect suspicious electricity consumption patterns.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Deployment
The model was trained and executed in a cloud-based Google Colab environment.

## Result
The system successfully identified abnormal electricity usage cases using anomaly detection.
